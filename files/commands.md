# Команды

---

# Важные

/login `<пароль>` - авторизация на сервере.<br>
/register `<пароль>` `<пароль>` - регистрация на сервере.<br>
/changepassword `<пароль>` `<пароль>` - смена пароля на сервере.<br>
/preium - включить режим `лицензии`.<br>

> [!DANGER] ВАЖНО
> Данный режим рекомендуется использовать если у вас есть Лицензия.

---

# Чат

`сообщение` - локальный чат<br>
!`сообщение` - глобальный чат<br>

/msg `ник` `сообщение` - отправить личное сообщение `игроку`<br>
/reply `сообщение` - ответить на последнее личное сообщение<br>

---

# Позы

/sit - сесть<br>
/sit toggle - переключает возможность сидеть для игрока<br>

/lay - лечь<br>
/craw - ползти<br>
/spin - крутиться<br>

---

# Музыкальные диски

> [!WARNING] ВАЖНО
